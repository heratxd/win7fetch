# 🚀 win7fetch

Привет! **win7fetch** — это очень легкий, реактивно быстрый и просто красивый аналог знаменитой утилиты `neofetch`, написанный специально под Windows. Он без проблем заведется как на древней Windows XP или семерке, так и на современных Windows 10 и 11.

Я написал этот проект специально для слабых компьютеров и ноутбуков (например, если у тебя всего 512 МБ оперативки). Он выдает всю инфу о системе мгновенно и не заставляет твое железо пыхтеть!

---

## 🔥 Почему он круче обычного neofetch на Windows?

1. **⚡ Работает со скоростью света**:
   Обычные порты `neofetch` или стандартная команда `systeminfo` на слабом ноуте могут затупить секунд на 15–20, мучая жесткий диск. `win7fetch` забирает большинство данных напрямую из реестра Windows (`reg query`), поэтому он запускается и выдает результат **меньше чем за полсекунды**!
   
2. **🧠 Кушает сущие копейки (меньше 2–3 МБ памяти)**:
   Вся утилита — это один простой пакетный файл `.cmd` и крошечный скрипт-помощник на VBScript. Никакого тяжелого .NET, Java или PowerShell, которые сожрали бы половину твоей оперативки при запуске. 

3. **🎨 Официальные логотипы и умные цвета**:
   Утилита сама понимает, на какой Windows запущена, и рисует **оригинальные ASCII-логотипы прямо из исходников neofetch**:
   - **Windows XP / Vista / 7** — тот самый легендарный волнистый флаг. В цвете он разделен на 4 каноничных лепестка (Красный, Зеленый, Синий, Желтый) с сохранением культовой красной точки-акцента.
   - **Windows 8 / 10** — современный скошенный плиточный логотип в перспективе.
   - **Windows 11** — каноничный ровный квадратный логотип.
   
   **И самое главное:** утилита сама проверяет, поддерживает ли твоя консоль цвета (ANSI). Если ты запустишь её в стандартной старой командной строке Windows 7, она не будет заваливать экран мусором вроде `←[91m`, а аккуратно выведет красивый монохромный логотип.

4. **📦 Установка в один клик (права админа не нужны!)**:
   Тебе не нужно лезть в настройки системы. Просто введи одну команду, и утилита сама пропишется в PATH твоего пользователя.

## 🛠️ Параметры, с которыми можно играть

Скрипт отлично работает сам по себе, но у него есть полезные ключи:

* `-h`, `--help` — Показать эту простую справку.
* `-c`, `--color` — Принудительно включить цвета. Пригодится, если ты сидишь через продвинутый терминал на семерке (например, ConEmu, Git Bash или с установленным ANSICON).
* `-n`, `--no-color` — Принудительно отключить цвета и вывести классический монохромный ASCII-арт.
* `-i`, `--install` — Установить утилиту в систему.
* `-u`, `--uninstall` — Полностью удалить утилиту из системы и очистить пути.

## 📦 Как установить себе на компьютер?

1. Перенеси файл `win7fetch.cmd` к себе на компьютер.
2. Открой командную строку (`cmd`) в папке, куда закинул файл, и напиши:
   ```cmd
   win7fetch --install
   ```
3. Перезапусти командную строку (или терминал), чтобы применились новые пути.
4. Готово! Теперь из абсолютно любой папки на компьютере ты можешь написать просто:
   ```cmd
   win7fetch
   ```

## ⚙️ А что у него под капотом? (Для тех, кому интересен код)

Для идеального баланса скорости и легкости утилита разделяет работу на два этапа:
* **Статичные данные** (процессор, модель материнки/ноута, версия Windows, версия ядра, оболочка) считываются напрямую из реестра Windows через быстрые запросы `reg.exe`.
* **Динамичные данные** (оперативка, активная видеокарта, текущее разрешение экрана и время работы uptime) собирает крошечный VBScript-помощник, который пишется в системную папку `%TEMP%` прямо во время запуска, быстро отрабатывает через встроенный WMI и тут же удаляется, не оставляя следов.
* **Uptime (время работы)** вычисляется полностью локально через встроенные математические функции VBScript `DateSerial` и `TimeSerial`. Скрипт не парсит строки даты (что часто приводило к вылетам на разных языковых версиях Windows), а математически считает разницу во времени. Поэтому он никогда не заглючит, будь у тебя русская, английская или любая другая винда.
